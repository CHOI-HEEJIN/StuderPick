# StuderPick 

스터디카페 통합 예약 웹사이트
<ul>
<li>전국 스터디카페의 스터디룸 또는 시간제 이용권을 한번에 예약 및 결제하는 웹사이트</li>
<li>스터디그룹 개설 및 참여, 맛집 찾기, SNS 기능 제공</li>
<li>Chart.js, Chartist를 이용한 카페 운영자의 수익 현황 시각화</li>
</ul>
